Background

About Me

I am a Mechanical Engineering graduate from GCE Erode who chose to build with code as much as with hardware. That decision led me to develop production-grade IoT systems, industrial web applications, and embedded firmware for real manufacturing clients.

At Sri Kalaivani Industries, I built an RFID attendance system using ESP8266 that eliminated 90% of manual work, developed a web-based HMI for a Special Purpose Machine communicating over WLAN, and implemented LoRa-based remote communication between two units using SX1278 and STM8S003F3.

My background spans Embedded C, Python, PHP, HTML/CSS, SQL, and hardware platforms including Raspberry Pi, Arduino, and NVIDIA Jetson TX2. I understand systems end-to-end — from GPIO and I2C registers to REST APIs and web dashboards.
